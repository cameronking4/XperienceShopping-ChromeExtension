Xperience Shopping Plugin (v1)

This site hosts Xperience Shopping size recommendation plugin. It takes input to create a virtual avatar and calculates body measurements. Simply inject this plugin into your website.

The plugin is used by placing the script in head of js site to popup Xperience Shopping wizard plugin. Click the button below to demo the plugin

Place into head of your site:

<script href="https://xperience-shopping.netlify.app/dist/xperience-plugin.js">

Call the Xperience Shopping function

<button onclick="launchXperience()">Launch Xperience Plugin</button>
